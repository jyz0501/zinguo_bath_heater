# zinguo_bath_heater
峥果智能浴霸homeassistant 云端插件，支持 hacs

## 安装方法

### 通过 HACS 安装
1. 在 HACS 中点击 "集成" -> "三个点" -> "自定义存储库"
2. 添加存储库 URL: `https://github.com/jyz0501/zinguo_bath_heater`
3. 类别选择 "集成"
4. 点击 "添加"
5. 在 HACS 中搜索 "峥果智能浴霸" 并安装
6. 重启 Home Assistant
7. 在 "配置" -> "设备与服务" -> "集成" 中点击 "添加集成"，搜索并添加 "峥果智能浴霸"

### 手动安装
1. 下载最新版本的 ZIP 文件
2. 将 `custom_components/zinguo_bath_heater` 目录复制到 Home Assistant 的 `custom_components` 目录下
3. 重启 Home Assistant
4. 在 "配置" -> "设备与服务" -> "集成" 中点击 "添加集成"，搜索并添加 "峥果智能浴霸"

## 配置说明

1. 输入您的峥果智能浴霸账号和密码
2. 选择电机版本（单电机或双电机）
3. 设置轮询间隔（默认30秒）

## 功能支持

- 照明控制
- 吹风控制
- 换气控制
- 取暖1控制
- 取暖2控制
- 全关功能
- 温控保护
- 多种传感器数据
- 定时功能

## 注意事项

- 该插件通过云端API控制设备，需要设备保持网络连接
- 建议使用稳定的网络环境以确保控制指令及时送达
- 轮询间隔不宜设置过短，建议10-60秒之间
